A guide to Placid Slider WP Plugin

Placid Slider is a customizable WordPress slider plugin that will help you add smootly scrolling horizontal slideshow to your WordPress site's front page, other pages, posts, sidebar widgets etc.

The posts, pages and the media library images can be easily added to the slider thru the edit panel on the WordPress admin.

The slider can be easily inserted anywhere on the site either using a template tag in the respective template file or using Placid Slider shortcode or using a simple widget. Also you can create multiple sliders, so that you can display different content in slider on different pages. And the best thing is, you can create multiple settings as well, and while embedding the slider you can specify which setting you would need for a particular instance of slider.

The slider can be very easily customized using the easy admin panel options. If you want to change the design of the slider with minute details, you can do that through CSS, so that you can create your own styles and change the looks of the slider as you wish.

How to install the plugin

  1. Uzip the placid-slider.zip folder.
  2. Upload the placid-slider folder to your /wp-content/plugins directory.
  3. In your WordPress dashboard, head over to the Plugins section.
  4. Activate Placid Slider.

Basic steps to implement the Placid Slider

  1. Activate the plugin.
  2. Add the posts,pages and/or media library images to the slider. You would get 'Add to Placid Slider' checkbox on the edit panel (of post/page/image) after plugin activation.
  3. Go to slider settings -> Change the width of the slider and the height of the Scrooling Area, depending on your theme width and your requirement. Also, remeber to change the width and height of the thumbnail Image on Placid Slider Settings Page => Slider Content tab => Thumbnail Image. You can change other parameters like colors, borders, image location and whether to show the post content or not etc.
  4. Now you can insert the Placid Slider on your WordPress site using either the template tag or the shortcode or the widget as follows.

Placid Slider Template tag

<?php if ( function_exists( 'get_placid_slider' ) ) {
  get_placid_slider(); } ?>

In case you use multiple sliders option,

<?php if ( function_exists( 'get_placid_slider' ) ) {
  get_placid_slider('2'); } ?>

Where '2' is the slider ID, which you can get from the 'Sliders' settings panel.

Placid Slider Shortcode

Put the below shortcode in the post/page editor to embed the Placid Slider in the post/page content.

[placidslider]

In case you use multiple sliders option,

[placidslider id='2']

Where '2' is the slider ID, which you can get from the 'Sliders' settings panel.

Placid Slider Widget

You have got a 'Placid Slider Simple' widget on Widget administration panel. Just insert the widget in any of the active widget of your theme.

Placid Slider Category Template tag

You can use the below template tag to display the posts under particular category in chronologically reverse order:-

<?php if ( function_exists( 'get_placid_slider_category' ) ) {
  get_placid_slider_category($catg_slug); } ?>

Real World Example:

<?php if ( function_exists( 'get_placid_slider_category' ) ) {
  get_placid_slider_category('social-media'); } ?>
where 'social-media' (don't forget the quotes) is the slug of the category from which you wish to pull the posts in reverse chronological order (latest is shown first).

Placid Slider Recent Posts Template tag

You can use the below template tag to display the recent posts added to your WordPress site/blog in chronologically reverse order:-

<?php if ( function_exists( 'get_placid_slider_recent' ) ) {
  get_placid_slider_recent(); } ?>

'Settings' Admin Panel

The Settings fall under three sections:

  1. Basic Settings Tab
  2. Slider Content
  3. Navigation Settings

Basic Settings Tab

Basic Settings

You can set the Scrolling Speed, Maximum number of posts (slides) in the slider, background color, complete slider width, width and height of each item, border settings here.

Scrolling Speed: This is the speed of the scroller. It can be a numeric value > 0. If set to decimal value, the plugin will break.

Maximum Number of Posts in the Placid Slider : How many posts/pages or media images can be there in the Slider at max. Say if you add 10 media gallery images to the Slider and this setting is set to 5, then only 5 images will be shown in the Slider. Default setting is 10.

Complete Slider Width : If you enter 0, the slider will take the width of the container. If you enter numeric value greater than 0, it would be the width of the slider. I would suggest you to enter the actual slider width in this field rather than keeping it 0.

Scrolling Area Height (Excluding Slider Title) : Height of the Scrolling area of the Slider. If you choose to show Slider title like 'Featured Sponsors' or so through the settings panel, this height will not include the Slider Title's height.

Slide (Item) Width : Width of each item in the Slider. Should be specified, and should be > 0.

Other than the above, you can set the background color for each Slide item, border for each item and border color as well through the settings panel.

Miscellaneous

Retain these html tags : You can retain links and br tags in the slider posts. Against this settings, put all the html tags that you wish to retain in the 'Retain these html tags' input field. Put the tags in the following manner,

<br><a><p>

Do not separate them using commas, neither use ⁄ anywhere.

Minimum User Level to add Post to the Slider : This seeting indicates which users can add or remove the posts/pages/media images to the slider. You can give the permission to editors but not authors or keep the access restricted to the 'Administrator' from here.

Randomize Slides in Slider : You can randomize the slides added to the slider, so that everytime the page on which slider is embedded loads, the slides would appear in random order to the reader.

Text to display in the JavaScript disabled browser : If the Javascript support is disabled or JS is not supported by the visitor's browser, this text would be displayed to the visitor saying that here is a slideshow which you can view in proper format if you enable the JS support on your browser.

Placid Slider Styles to Use on Other than Post/Pages : You can creae your own styles for Placid Slider. From here you can select which stylesheet should be used on general templates (i.e. like index.php or category.php or archives.php) of your site (other than single posts and pages). For the sliders embedded on posts and pages, the stylesheet specified in this settings field would be default stylesheet, but you can individually select another stylesheet from the edit post/page panel for that specific post/page.

Multiple Slider Feature : Tick this box to enable the switch to create and embed 'multiple sliders'. Once this checkbox is ticked, you would get an option to create a new slider on the 'Sliders' admin page under 'Placid Slider' settings. Also, you can add posts to various created sliders and embed different sliders on different pages.

Enable FOUC : In case you tick this option, while the page with the slider is loaded, you can see the Flash of Unstyled content (i.e. the content in the slider will appear without any styling while the web page is loading). This box is uncheked by default, which means the complete slider with its content will be loaded at once when the complete DOM elements (html page) is ready and loaded.

Custom Styles : You can add your own custom CSS for each of the Sliders created or to override the theme styles.

#slider_id_1 .sldr_title{background:#cccccc} 
#slider_id_2 .sldr_title{background:#ff0000;color:#fff;}

Slider Content

Slider Title

Here you can specify the main title of the slideshow, like 'Featured' or 'Recommended' or so on. By default it is 'Featured Slides'. Also you can change the font, color and looks of this title thru these settings.

Post Title

These settings control the look of the title of the content in each slide.

Thumbnail Image

These settings apply to the images extracted for the larger slides on the left hand section of the slider.

Image Pick Preferences : This setting indicates in what priority you want the images extracted for the post or the page. There are four entries, if all the checkboxes are checked that means the post/page will be scanned for the images in following sequence:

  1. Looks for an image by custom field. By default the custom field is slider_thumbnail. You can change its value, may be if you want to use the 'thumbnail' custom field.
  2. If no image is added by custom field, check for a post image/featured image set for the post (WordPresss 2.9+ feature).
  3. If no image is found, it grabs an image attached to your post.
  4. If no image is attached, it can extract an image from your post content.

Say if you uncheck the 'Consider Images attached to the post ... Order of the Image attachment to pick ' checkbox, it will skip the step 3 above.

Say, if you do not want the scanner to look to thumbnails and featured image and just directly pick the image from the post content, just untick the first three boxes and keep the fourth box checked.

Other settings include Image extract size, Display Image Size (width and height of the image) and border settings.

You can tick the check box saying 'Make pure Image Slider' in case you want only images in the slides and no content.

Slide Content

Show content in slides below title : Checking this box will enable the content/excerpt to be displayed below the title of the slide item. By default this is OFF.

You can customize the looks, font, color etc. of the Slide Content area here.

You can choose how you would want the slider to pull the content i.e. 'From Post Content', 'Post Excerpt' or 'Slider Content Custom Field'. If you set it to 'Slider Content Custom Field', you should assign the value to 'slider_content' custom field as the content for the slide.

Maximum content size : You can specify the maximum content size in the slide either in number of characters or in number of words. The 'number of words' value if entered will override the 'number of characters' value. So if you want the maximum content size in 'number of characters' leave the 'numer of words' setting to blank (do not enter any value in it).

Navigation Settings

You can enable the Prev/Next or Left/Right Navigation buttons here. If enabled, it will disable to 'Auto Scrolling' or 'Auto Sliding' of the slides.

'Sliders' Admin Panel

Through this panel, you can remove the post/page/media image added to the slider directly or completely empty the slider. If the 'Multiple Sliders Feature' is enabled (it is enabled by default), you can create new sliders from this panel and delete the sliders.

Also, you can reorder the entries in the slider thru this settings panel.

FAQs

How to convert the Placid Slider into Pure Image Slider?

If you do not want the Post Title/Image title to appear in the Slide over the image, you can definitely convert the Placid Slider into a pure Image Slider. Go to Placid Slider Settings => Slider Content Tab => Thumbnail Image => Tick the box saying 'Make pure Image Slider'. Save the Settings.

Images are not getting displayed in Placid, how to fix??

Placid Slider uses TimThumb Image cropping. On some servers the timthumb.php cannot crop the images and move it to cache folder due to server permissions. Due to which, the images are not visible in the Placid Slider.

In case you are getting this issue, you would need to change the Permissions to two folders to make it work. Go to wp-content => plugins => placid-slider => includes => There will be two folders named 'cache' folder and 'temp' folder. Change the permissions of these folders to 777 (sometimes 755 also works).

In case the issue still exists, please go through this small debugging guide for timthumb image cropping.
How to create Multiple Settings?

On the 'Settings' admin panel of Placid Slider, at the top you would see 'Create New Settings Set' button. Click the button, you would be asked to confirm that you need to create a new settings set

Once you create a new settings set, you would still be served the current settings. You can access and modify the new settings using the settings link on the right sidebar named 'Setting Panels'.

Before modifying the settings, do check on which settings panel you currently are. You can check this using the title of the Settings page which would be such as 'Placid Slider Settings Default', 'Placid Slider Settings 2' and so on

How to apply various settings to the Sliders?

Placid Slider Template tag

<?php if ( function_exists( 'get_placid_slider' ) ) {
  get_placid_slider($slider_id='1',$set='2'); } ?>

In case you use multiple sliders option,

<?php if ( function_exists( 'get_placid_slider' ) ) {
  get_placid_slider($slider_id='2',$set=''); } ?>

Where '2' is the slider ID, which you can get from the 'Sliders' settings panel.

In case you enter '' (empty) for the $set, it will take the default settings.

Placid Slider Shortcode

Put the below shortcode in the post/page editor to embed the Placid Slider in the post/page content.

[placidslider set='2']

In case you use multiple sliders option,

[placidslider id='2']

[placidslider id='2' set='3']

Where '2' is the slider ID, which you can get from the 'Sliders' settings panel and '3' is the ID of the settings created.

How to create different stylesheets for Placid Sliders?

You should have FTP access to your site's folder in order to create your own styles.

Go to wp-content => plugins => placid-slider folder => css => skins

Create a new folder with a name say... 'custom'. Copy the content of 'sample' folder into 'custom' folder. Now go to custom folder => > style.css file. Open it with Notepad or Notepad++ or the editor you use to edit CSS files, add and modify/remove your own CSS rules. Save the CSS file. Upload the complete custom folder to the server, inside the skins folder.

Now on the Placid Slider settings page => go to 'Miscellaneous' settings => You would get value as 'custom' in dropdown for the setting 'Placid Slider Styles to Use on Other than Post/Pages'. Select 'csutom' and save the settings.

As well, on Post/Page edit panel, you would get an option saying 'Stylesheet to use if slider is displayed on this Post/Page ' under 'Placid Slider' custom meta box. Select the value as 'custom' and save the post/page.

You can create any number of style folders as 'custom' and name them whatever you want.

How to add media library images to the slider?

Go to Edit Media thru Media => Library and open the image in edit mode. You would get an option to add the media image to the Placid Slider, you can assign the link URL to which the lside will link and you can also choose not to link a particular image to any URL.

How to to assign different 'Slider Title' for various sliders created?

In case you select multiple sliders option, you are able to create and embed different sliders with different content on various pages of your WordPress site. You may wish to have different Slider Titles for different sliders created (mostly when you are inserting the slider thru Placid Slider Widget). Yes, you can do that!!

When you create a new slider, you can give different names to different sliders created, which can be used as slide title. Go to Placid Slider 'Settings' => Slider Title settings => Pick Slider Title From => and select 'Slider Name' for the settings. Save the settings.

Also if you are using Category Widget or Reent Posts Slider Widget for Placid, you would get an option to enter the Slider title of your choice.

How to link the slide to different URL than the post/page permalink?

On the Edit Post/Page/Media image panel under Placid Slider fields, you would get an input field named 'Slide Link URL', you can input whatever URL you want the slide to link to in this field. Save the entry.

How to disable the linking of the slide to any URL?

On the Edit Post/Page/Media image panel under Placid Slider fields, you would get an checkbox named 'Do not link this slide to any page(url)', you can uncheck the box to choose not to link the slide to any url. Save the entry.

How to disable the promotional ads on the settings page?

Go to Placid Slider 'Settings' => Miscellaneous settings => Show Promotionals on Admin Page => and select 'No' for the settings. Save the settings.
