Jeromes Keywords, Live-Vorschau, Kommentar-Quciktags, Kommentar-Smileys (WP-Grins), Theme Switcher und Pagebar.

Yes, you can change this Theme and use it also for a commercial Project.

This Theme is ready für Sidebar-Widgets. For some Widgets its maybe necesary to make a little changes on the Layout. 

In this Theme this Plugins are "integrated"**: 
Jeromes Keywords, Live Preview, Comment Quicktags, Comment Smileys (WP-Grins), Theme Switcher and Pagebar.
